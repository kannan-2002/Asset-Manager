## üíª Local Development Setup

Follow these steps to clone the repository and run the project on your local machine.

### Prerequisites

* **Node.js & npm** (Node Package Manager) installed on your system.

### Installation and Run

1.  **Clone the Repository**

    Open your terminal and clone the project, navigating into the directory:
    ```sh
    # Step 1: Clone the repository using the project's Git URL.
    git clone https://github.com/kannan-2002/Asset-Manager.git

    # Step 2: Navigate to the project directory.
    cd Asset-Manager
    ```

2.  **Install Dependencies**

    Use npm to install all necessary project dependencies:
    ```sh
    # Step 3: Install all packages.
    npm install
    ```

3.  **Start the Development Server**

    Launch the development server. The application will be available in your browser.

    ```sh
    # Step 4: Start the development server (uses Vite).
    npm run dev
    ```
